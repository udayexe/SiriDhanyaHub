# 🌾 Siri Dhanya Hub
### Empowering Karnataka’s Millet Value Chain

Siri Dhanya Hub is an Android application built to support Karnataka’s growing millet ecosystem by connecting farmers, consumers, and health-conscious communities through technology.

The platform promotes sustainable agriculture, fair trade, and healthy living by making millet-related information, recipes, pricing, and direct purchasing accessible in one secure mobile experience.

---

## 📱 Features

### 📊 Mandi Watch
Get live market price updates for different millet varieties such as:
- Finger Millet (Ragi)
- Pearl Millet (Bajra)
- Barnyard Millet
- Foxtail Millet
- Little Millet

This helps farmers make informed selling decisions and improves market transparency.

---

### 🍲 Recipe Lab
Discover traditional and modern millet recipes designed to:
- Encourage healthier eating habits
- Promote Karnataka’s food culture
- Make millets easier to include in daily diets

---

### 💪 Health Benefits
Learn about the nutritional and health advantages of millets including:
- High fiber content
- Low glycemic index
- Rich minerals and proteins
- Climate-friendly food source

---

### 🛒 Buy Direct
Connect directly with:
- Farmer Producer Organizations (FPOs)
- Local sellers
- Millet producers

This ensures:
- Fair pricing
- Fresh produce
- Better farmer income

---

## 🔐 Security & Privacy

Siri Dhanya Hub focuses heavily on secure user experience.

### Security Features
- 🔑 4-digit PIN Authentication
- 👆 Fingerprint Authentication
- 😀 Face Unlock Support
- 🔒 Encrypted Local Storage
- 🛡️ Android Security Crypto Integration

Sensitive user data is protected using industry-standard Android security practices.

## 🚀 Getting Started

Make sure you have the following installed:
- Android Studio
- JDK 17+
- Gradle
- Android SDK

---

### Installation

Clone the repository:

```bash
git clone https://github.com/udayexe/SiriDhanyaHub.git
```

Open the project in Android Studio.

Sync Gradle dependencies and run the application on:
- Android Emulator
- Physical Android Device
